# Great Tunb Island NFT

## Overview

Great Tunb Island is a geographic NFT representing the island as a unique digital real-world asset.

The NFT is designed to be associated with higher-level ownership structures within the ecosystem.

---

## Technical Specifications

- ERC-721
- Solidity 0.8.27
- ERC721URIStorage
- ERC721Royalty
- ERC721Pausable
- ERC721Burnable
